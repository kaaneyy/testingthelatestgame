# Pawn & Pick

A fast, touch-friendly React game about running a tiny music pawn shop. Buy stock, place instruments, serve customers, hire helpers, and grow the store across days.

## Run Locally

```bash
npm install
npm run dev
```

## Deploy On Vercel

Import this repository in Vercel and keep the defaults:

- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`

The included `vercel.json` pins those settings for clean deployments.
